六、Todo List

小写 x 表示已完成， 空格表示未完成

  • 支持数学公式
  • 支持TodoList
  • 支持自定义样式
  • 支持自定义图床

    七、粗体，斜体，高亮和删除线

    使用 ** 表示粗体。 使用 * 或者 _ 表示 斜体。 使用 * 表示 *全部粗体和斜体*** 使用 == 表示 ==高亮==
    ❗ 部分编辑器不支持高亮 使用表示 ~删除~~

八、列表

使用 1. 1. 1. 表示有序列表，使用 * 或 - 或 + 表示无序列表。

无序列表或有序列表标记和后面的文字之间要有一个空格隔开。

有序列表标记不是按照你写的数字进行显示的，而是根据当前有序列表标记所在位置显示的，如示例 1 所示。

无序列表的项目符号是按照实心圆、空心圆、实心方格的层级关系递进的，如例 2 所示。通常情况下，同一层级使用同一种标记表示，便于自己查看和管理。

九、分割线

使用 — 或者 *** 或者 * * * 表示水平分割线。

  1. 只要 * 或者 - 大于等于三个就可组成一条平行线。
  2. 使用 — 作为水平分割线时，要在它的前后都空一行，防止 --- 被当成标题标记的表示方式。
